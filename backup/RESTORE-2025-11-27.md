# Restore / Use backup

Archivos generados:
- backup-full-2025-11-27.bundle : Git bundle conteniendo todas las referencias locales (ramas, tags)
- backup-full-2025-11-27.zip : Zip del snapshot del branch `backup/full-2025-11-27`

Para restaurar y enviar al remote `advanced` (app-misional-avanzada):

1) Con el bundle:
```powershell
# clona el bundle en un nuevo directorio
git clone backup/backup-full-2025-11-27.bundle repo-backup
cd repo-backup
# agrega el remote (HTTPS o SSH) según configuración
# SSH example (requires SSH key):
git remote add advanced git@github.com:Victordaz07/app-misional-avanzada.git
# OR HTTPS example (requires PAT or auth):
# git remote add advanced https://github.com/Victordaz07/app-misional-avanzada.git
# push the backup branch
git push advanced HEAD:refs/heads/backup/full-2025-11-27
```

2) Con el ZIP (no mantiene la información de Git, solo archivos):
```powershell
mkdir repo-backup-zip
Expand-Archive -Path backup/backup-full-2025-11-27.zip -DestinationPath repo-backup-zip
cd repo-backup-zip
# Init git, make initial commit
git init
git add .
git commit -m "restore: snapshot from 2025-11-27"
# push to remote advanced
git remote add advanced git@github.com:Victordaz07/app-misional-avanzada.git
git push advanced master:refs/heads/backup/full-2025-11-27
```
