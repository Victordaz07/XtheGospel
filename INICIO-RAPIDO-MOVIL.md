# 🚀 Inicio Rápido - Ver la App en tu Móvil

## ⚡ Método Rápido (3 pasos)

### 1️⃣ Inicia el servidor
Abre una terminal y ejecuta:
```bash
npm run dev
```
Espera a ver: `Local: http://localhost:3001/`

### 2️⃣ Crea el túnel público
Abre una **nueva terminal** (no cierres la anterior) y ejecuta:
```bash
lt --port 3001
```
Verás algo como:
```
your url is: https://xxxx-xx-xxx-xxx-xxx.loca.lt
```

### 3️⃣ Abre en tu móvil
- Copia la URL completa (ejemplo: `https://xxxx-xx-xxx-xxx-xxx.loca.lt`)
- Ábrela en el navegador de tu móvil
- **¡Funciona desde cualquier lugar, sin WiFi!** 📱

---

## 📋 O usa el script automático (Windows)

Simplemente ejecuta:
```bash
iniciar-con-tunel.bat
```

---

## ⚠️ Notas importantes

- ✅ La primera vez que abras la URL, te pedirá permisos - acepta para continuar
- ✅ La URL cambia cada vez que reinicias el túnel
- ✅ Mantén ambas terminales abiertas mientras uses la app
- ✅ Para detener: Presiona `Ctrl+C` en ambas terminales
